# ProjetoCSharp_BaseDados

Sistema de gestão escolar desenvolvido em C# com Windows Forms,
com ligação a base de dados MySQL.

## Funcionalidades
- Autenticação de utilizadores (login)
- Gestão de alunos e cursos
- Interface gráfica em Windows Forms

## Tecnologias utilizadas
- C# / .NET Framework 4.7.2
- Windows Forms
- MySQL (via MySql.Data)
- Visual Studio

## Configuração
1. Clona o repositório
2. Cria a base de dados usando o ficheiro `SQL_Projeto.sql`
3. Copia `App.config.example` para `App.config` e preenche com os teus dados de ligação
4. Abre o projeto no Visual Studio e corre
